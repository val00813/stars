Stars
This project is a creative coding sketch made using TouchDesigner.

Description
The work explores the position of women within historical narratives—they are not absent, but have long been ‘filtered out’ and obscured by unequal social power structures. Through digital media, I have transformed this abstract social phenomenon into a dynamic particle system: only when the viewer’s gaze and engagement (gestural interaction) intervene do those ‘fragments of women’s history’ hidden in the shadows come to life, illuminated once more, revealing their original brilliance.
